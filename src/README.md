# 💰 Expense Tracker App

A full-stack Expense Tracker built using React, Node.js, Express, and MongoDB.

## 🚀 Features
- User Signup & Login (JWT Authentication)
- Add, Delete, Clear Transactions
- Real-time Balance Calculation
- Monthly Expense Analysis (Bar Chart)
- Income vs Expense (Pie Chart)
- Category-wise Expense Breakdown
- Clean Dashboard UI
- Toast Notifications for user feedback

## 🛠 Tech Stack
- Frontend: React, Axios, CSS
- Backend: Node.js, Express
- Database: MongoDB Atlas
- Charts: Recharts
- Authentication: JWT + bcrypt

## 🌐 Deployment
- Frontend: Vercel
- Backend: Railway
- Database: MongoDB Atlas

## 📊 Features in Detail
- Secure login system using JWT
- RESTful APIs for transaction management
- Interactive charts for financial insights
- Responsive dashboard layout
